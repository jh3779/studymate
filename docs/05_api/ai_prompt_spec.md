# AI 프롬프트 명세서

## 1. 문서 목적

이 문서는 StudyMate에서 사용할 AI 요약 생성 프롬프트와 AI 퀴즈 생성 프롬프트의 입력, 출력, 검증 기준을 정의한다.

## 2. 핵심 내용

AI 응답은 앱에서 파싱 가능한 JSON 형식이어야 한다.

| 기능 | 입력 | 출력 |
| --- | --- | --- |
| 요약 생성 | 사용자 학습 내용 | summary 배열, keywords 배열 |
| 퀴즈 생성 | 요약 또는 원문 내용 | 문제 객체 3개를 가진 JSON 배열 |

## 3. 상세 설명

### 요약 생성 프롬프트

```text
너는 학습 내용을 요약해주는 AI 학습 도우미다.

아래 학습 내용을 바탕으로 다음 형식에 맞게 결과를 생성해라.

조건:
- 핵심 요약은 3~5줄로 작성
- 어려운 표현은 쉽게 풀어서 작성
- 중요한 키워드 3개를 추출
- 결과는 JSON 형식으로 반환

입력 내용:
"""
{사용자 학습 내용}
"""

출력 형식:
{
  "summary": [
    "요약 문장 1",
    "요약 문장 2",
    "요약 문장 3"
  ],
  "keywords": ["키워드1", "키워드2", "키워드3"]
}
```

### 요약 응답 검증 기준

| 필드 | 타입 | 조건 |
| --- | --- | --- |
| summary | array<string> | 3~5개 권장, 최소 1개 이상 필요 |
| keywords | array<string> | 3개 권장 |

### 퀴즈 생성 프롬프트

```text
너는 시험 대비용 객관식 문제를 만들어주는 AI 학습 도우미다.

아래 학습 내용을 기반으로 객관식 문제 3개를 생성해라.

조건:
- 문제는 4지선다형
- 정답은 1개만 존재
- 각 문제마다 해설 포함
- 너무 쉬운 문제만 만들지 말 것
- 결과는 JSON 배열 형식으로 반환

입력 내용:
"""
{요약 또는 원문 내용}
"""

출력 형식:
[
  {
    "question": "문제 내용",
    "options": ["보기1", "보기2", "보기3", "보기4"],
    "answerIndex": 0,
    "explanation": "정답 해설"
  },
  {
    "question": "문제 내용",
    "options": ["보기1", "보기2", "보기3", "보기4"],
    "answerIndex": 2,
    "explanation": "정답 해설"
  }
]
```

### 퀴즈 응답 검증 기준

| 필드 | 타입 | 조건 |
| --- | --- | --- |
| question | string | 비어 있으면 안 됨 |
| options | array<string> | 정확히 4개 |
| answerIndex | number | 0 이상 3 이하 |
| explanation | string | 비어 있으면 안 됨 |

## 4. 개발 시 참고사항

- AI 응답 앞뒤에 설명 문장이 섞일 수 있으므로 JSON 추출 로직이 필요할 수 있다.
- 앱은 JSON 파싱 실패 시 사용자에게 재시도 안내를 제공한다.
- AI 모델은 학습 보조 용도이며, 생성 결과의 정확성은 사용자가 최종 확인해야 한다.
- 프롬프트는 MVP에서 고정하고, 추후 과목별 난이도 조절 기능을 확장할 수 있다.

## 5. 확인 체크리스트

- [ ] 요약 프롬프트가 3~5줄 요약과 키워드 3개를 요구하는가?
- [ ] 퀴즈 프롬프트가 4지선다와 해설을 요구하는가?
- [ ] 응답 JSON 필드명이 모델 설계와 일치하는가?
- [ ] answerIndex가 0부터 시작한다는 기준이 명확한가?
- [ ] JSON 파싱 실패 대응이 고려되어 있는가?
